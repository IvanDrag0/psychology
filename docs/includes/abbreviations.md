*[AI]: Artificial Intelligence — computer systems designed to perform tasks that typically require human intelligence
*[ML]: Machine Learning — computers learning patterns from data rather than being explicitly programmed
*[NLP]: Natural Language Processing — teaching computers to understand, interpret, and generate human language
*[EMA]: Ecological Momentary Assessment — repeated real-time sampling of behaviors and experiences in daily life
*[XAI]: Explainable AI — methods that make AI decisions interpretable and transparent to humans
*[LLM]: Large Language Model — AI systems trained on massive text datasets to understand and generate language (e.g., GPT-4, Claude)
*[CV]: Cross-Validation — a resampling method that tests how well a model generalizes to new data
*[AUC]: Area Under the Curve — a metric summarizing how well a classifier distinguishes between classes
*[ROC]: Receiver Operating Characteristic — a plot showing the trade-off between true positive rate and false positive rate
*[LASSO]: Least Absolute Shrinkage and Selection Operator — a regression method that shrinks some coefficients to zero for variable selection
*[PCA]: Principal Component Analysis — a dimensionality reduction technique that finds the axes of greatest variance
*[SVM]: Support Vector Machine — a classification algorithm that finds the optimal boundary between groups
*[SHAP]: SHapley Additive exPlanations — a method for explaining individual predictions by assigning each feature a contribution value
*[LDA]: Latent Dirichlet Allocation — a topic modeling technique that discovers hidden thematic structure in text
*[ED]: Eating Disorder — a range of psychological conditions involving persistent disturbance of eating behaviors
*[BDI]: Beck Depression Inventory — a widely used self-report measure of depression severity
*[CAT]: Computerized Adaptive Testing — assessments that dynamically select items based on prior responses
*[IRB]: Institutional Review Board — a committee that reviews and approves research involving human participants
*[API]: Application Programming Interface — a structured way for software programs to communicate with each other
*[BERT]: Bidirectional Encoder Representations from Transformers — a language model architecture designed for understanding text in context
*[GPT]: Generative Pre-trained Transformer — a language model architecture designed for generating text
*[DV]: Dependent Variable — the outcome variable being predicted or explained in a study
*[IV]: Independent Variable — a predictor variable hypothesized to influence the outcome
*[ICC]: Intraclass Correlation Coefficient — a reliability metric measuring agreement among raters or repeated measures
